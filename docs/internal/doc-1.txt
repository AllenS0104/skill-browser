# 🧠 一、什么是 Browser Skill？

在现代 **AI Agent 系统（2025\~2026）** 里：

Agent 已经不再是：

```text
LLM + Tool
```

而是：

```text
Cognition + Execution + Environment
```

其中：

👉 **Browser Skill = Agent 进入真实互联网世界的执行能力**

***

# 📈 二、Browser Skill 技术演进

***

## 🧱 第一代 Browser Automation（2022 之前）

典型技术：

*   Selenium
*   Puppeteer
*   Playwright

能力：

```text
✔ DOM 操作
✔ 点击按钮
✔ 填写表单
✔ XPath Selector
```

限制：

❌ 不理解 UI  
❌ React/SPA 页面失效  
❌ DOM 变化即崩  
❌ 无语义推理能力

本质：

> Blind HTML Robot

***

## 🧱 第二代 LLM Browser Agent（2023）

典型：

*   LangChain Browser Tool
*   AutoGPT Web Agent
*   OpenAI Early Browser Tool

运行模式：

```text
LLM → 查 Selector → 执行 click()
```

问题：

❌ Lazy Rendering  
❌ Shadow DOM  
❌ Virtual List  
❌ Captcha  
❌ SPA Hydration

因为：

> Web 是视觉交互系统  
> 而不是 DOM Tree

***

## 🔥 第三代 Vision Browser Agent（2024）

代表：

*   Claude Computer Use
*   GPT‑4o GUI Agent
*   Open Interpreter GUI Mode

运行逻辑：

```text
Screenshot
↓
Vision Encoder
↓
LLM reasoning
↓
Mouse Action
```

优点：

✅ 不依赖 selector  
✅ 理解界面布局

缺点：

🚧 Screenshot Loop → 效率低

# 🌐 九、什么是 fetch？

***

## ✅ 标准定义

```text
fetch = 程序级 HTTP 网络调用接口
```

用于：

👉 代码访问 Web API / SaaS 服务

***

## Web 原生 API 示例：

```js
fetch("https://api.shopify.com/orders", {
  method: "GET",
  headers: {
    "Authorization": "Bearer sk-xxx"
  }
})
.then(res => res.json())
.then(data => console.log(data));
```

***

## 等价 CLI 命令：

```bash
curl https://api.shopify.com/orders
```

***

# 🧠 十、fetch 在 AI Agent 中的作用
